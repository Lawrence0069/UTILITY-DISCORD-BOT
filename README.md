# 🤖 Discord Utility Bot

A powerful and customizable Discord bot built using Python.  
This bot includes moderation tools, ticket system, voice management, and more.

---

## 🚀 Features

- 🎫 Ticket System (Mod Mail)
- 👮 Role-based Moderation Commands
- 🔊 Voice Channel Management (move users, restrict access)
- 💬 AFK System with custom messages
- 📢 Auto messages (boost thank-you, suggestions guide)
- 🎵 Music Bot (optional)
- 🧠 MongoDB Database Support
- 🔒 Role-based Permissions (Junior & Senior Mods)

---

## 🛠️ Tech Stack

- Node.js (v20+)
- Discord.js
- MongoDB
- Express (optional for web features)
